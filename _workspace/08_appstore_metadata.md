## 2. 앱 이름 후보 + 부제목

### 후보 A: EngWrite (추천)
- **앱 이름**: `EngWrite - 매일 영작문`
- **부제목**: `AI가 원어민처럼 교정해주는 학습`
- **장점**: 영문 브랜드명이 직관적(Eng+Write), '매일'로 습관 학습 암시, '영작문'으로 검색 포착
- **단점**: 영문명이 비영어권에서 발음이 다소 생소할 수 있음

### 후보 B: 하루영작
- **앱 이름**: `하루영작 - 오늘의 3문장`
- **부제목**: `AI 원어민 교정으로 영어 작문 실력 UP`
- **장점**: '하루'+'영작'으로 한글 검색에 강함, '3문장'이 부담 없는 학습량 전달
- **단점**: 브랜드 확장성이 다소 제한적

### 후보 C: 쓰잉 (Write + ing)
- **앱 이름**: `쓰잉 - 영어 작문 트레이너`
- **부제목**: `한글 보고 쓰면 AI가 교정해줘요`
- **장점**: 2음절로 기억 용이, '쓰다'의 느낌과 영어 진행형(-ing) 조합이 세련됨
- **단점**: 브랜드 인지도를 처음부터 쌓아야 함, 검색 시 다른 결과와 혼동 가능성

## 3. 앱 설명문 (한국어, 4000자 이내)

```
한글 문장을 보고 영어로 써보세요.
AI가 원어민이 실제로 쓰는 표현으로 바로 교정해 드립니다.
하루 3문장이면 충분합니다.

━━━━━━━━━━━━━━━━━━━━━━

"영어 문장을 직접 써보고 싶은데, 내가 쓴 게 맞는지 누가 봐줬으면..."

EngWrite는 바로 그 고민에서 시작했습니다.

한글 문장이 주어지면 영어로 작문하고, AI가 즉시 원어민 수준의 교정과 점수를 알려줍니다. 번역기에 의존하지 않고, 직접 쓰면서 영어 감각을 키우는 가장 확실한 방법입니다.

▶ 이런 분에게 딱이에요

• 영어 문장을 직접 만들어보고 싶은 초급~중급 학습자
• 매일 조금씩 꾸준히 영어 실력을 쌓고 싶은 직장인/대학생
• 번역투가 아닌 자연스러운 영어 표현을 익히고 싶은 분
• 영어 작문 피드백을 받을 곳이 없는 독학러

▶ 핵심 학습 흐름

1. 테마를 고르세요 (일상영어 / 비즈니스영어)
2. 한글 문장을 보고 영어로 작문하세요
3. AI가 원어민 표현으로 교정 + 점수 + 상세 설명을 제공합니다
4. 핵심 표현 1개를 기억하세요 — 이것만으로도 실력이 쌓입니다

▶ 부담 없이, 그래도 확실하게

• 하루 3문장: 5분이면 끝나는 적정 학습량
• 힌트 단어: 막히면 핵심 단어를 펼쳐 확인 (스스로 접기/펼치기)
• 스와이프 카드 UI: 간편하게 문장을 넘기며 학습

▶ AI 교정이 다릅니다

일반 번역이 아닙니다. AI가 학습자의 문장을 분석하여:
• 원어민이 실제로 쓰는 자연스러운 표현으로 교정
• 틀린 부분을 하이라이트하고 왜 틀렸는지 한국어로 설명
• 대안 표현 1~2개를 추가 제안
• 10점 만점 채점으로 실력 변화를 눈에 보이게

▶ 동기 부여 시스템

• XP & 레벨: 작문할 때마다 경험치 획득, 레벨업의 성취감
• 연속 학습 스트릭: 매일 학습하면 스트릭이 쌓입니다
• 업적 배지: 첫 작문, 만점, 연속 7일 등 다양한 업적 달성
• 주간 리포트: 이번 주 학습량과 평균 점수를 한눈에 확인

▶ 복습으로 실력을 굳히세요

점수가 낮았던 문장을 모아 다시 도전하는 복습 모드를 제공합니다.
이전 점수와 비교하며 성장을 체감할 수 있습니다.

▶ 학습 기록을 놓치지 마세요

• 전체 교정 기록 저장 및 조회
• 날짜별, 테마별 학습 이력 확인
• 평균 점수 변화 추이 확인

━━━━━━━━━━━━━━━━━━━━━━

매일 3문장, 꾸준히 쓰면 영어가 달라집니다.
지금 시작해보세요.
```
