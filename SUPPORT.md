# 쓰잉 (Ssuing) 지원 센터

영어 작문 학습 앱 **쓰잉**을 사용해주셔서 감사합니다. 문제가 생기거나 문의하실 내용이 있다면 아래 안내를 확인해주세요.

---

## 📩 문의 / 지원 이메일

**ssuing.app@gmail.com**

다음 정보를 함께 보내주시면 더 빠르게 도와드릴 수 있습니다:
- 앱 버전 (설정 화면 하단에서 확인)
- iOS 기기 모델 / iOS 버전
- 문제 발생 화면 스크린샷
- 가능하면 재현 단계

영업일 기준 **1~2일 이내** 답변드립니다.

---

## ❓ 자주 묻는 질문

### 무료 체험은 어떻게 되나요?
- 7일간 모든 프리미엄 기능을 무료로 이용하실 수 있습니다.
- 체험 기간이 끝나기 24시간 전까지 해지하지 않으시면 자동으로 월 5,000원이 결제됩니다.

### 구독은 어떻게 해지하나요?
1. iPhone **설정** 앱 열기
2. 상단의 **본인 이름** (Apple ID) 탭
3. **구독** 선택
4. 목록에서 **쓰잉 프리미엄** 선택 → **구독 취소**

자세한 안내: [Apple 구독 관리 가이드](https://support.apple.com/ko-kr/118428)

### 결제 환불은 어떻게 받나요?
앱스토어 결제는 Apple이 처리합니다. [Apple Report a Problem](https://reportaproblem.apple.com)에서 환불 요청을 할 수 있습니다.

### 작문 교정이 이상해요
AI 기반 자동 교정이라 가끔 부자연스러운 결과가 나올 수 있습니다. 특정 문장의 교정 품질에 문제가 있다면 위 이메일로 해당 문장과 받은 결과를 보내주세요. 콘텐츠 개선에 반영합니다.

### 학습 기록이 사라졌어요
- 같은 Apple 계정으로 로그인했는지 확인해주세요.
- 앱을 삭제했다 재설치한 경우 일부 로컬 데이터가 사라질 수 있지만, 서버에 저장된 학습 기록은 다시 로그인하면 복구됩니다.
- 그래도 사라진 상태라면 위 이메일로 문의해주세요.

### 알림이 오지 않아요
1. iPhone **설정 → 알림 → 쓰잉**에서 알림 허용 확인
2. 앱 안 **설정 탭**에서 알림 토글 확인
3. 그래도 안 되면 앱 재시작 또는 재설치
