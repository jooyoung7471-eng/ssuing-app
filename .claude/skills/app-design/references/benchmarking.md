# 영어 작문 학습 앱 UX/UI 벤치마킹

## 1. 말해보카 (Malhaeboca)

### UI 특징
- 카드 기반 학습: 단어/문장을 카드 형태로 제시, 좌우 스와이프로 "알아요/몰라요" 분류
- 미니멀 디자인: 한 화면에 하나의 학습 요소만 집중 표시
- 진행률 바: 상단에 학습 진행률 시각화
- 다크/라이트 모드 지원

### 학습 플로우
1. 학습 세트 선택 → 2. 카드 학습 (스와이프) → 3. 퀴즈 → 4. 결과 요약

### 핵심 UX 패턴
- **스와이프 분류**: 좌/우 스와이프로 직관적 이진 분류
- **반복 학습**: 틀린 카드를 큐 뒤로 재배치
- **힌트 시스템**: 탭하면 예문/발음 펼치기
- 색상: 화이트 배경 + 포인트 블루(#4A90D9), 그린(성공), 레드(실패)

## 2. 듀오링고 (Duolingo)

### 작문/번역 UI
- 워드 뱅크: 주어진 단어 블록을 조합하여 문장 구성 (초급)
- 자유 입력: 직접 타이핑하여 번역 (중급 이상)
- 즉시 피드백: 제출 즉시 정답/오답 + 올바른 문장 표시
- 오답 하이라이트: 틀린 부분을 빨간색으로 강조

### 게이미피케이션
- **스트릭(연속 학습일)**: 매일 학습 동기 유지의 핵심
- **XP(경험치)**: 문제 정답 시 획득
- **리그/리더보드**: 주간 랭킹 경쟁
- **하트 시스템**: 오답 시 하트 소모, 소진 시 학습 중단
- **레벨업 애니메이션**: 완료 시 축하 애니메이션

### 일일 학습 플로우
1. 앱 진입 → 스트릭 확인 → 2. 레슨 선택 → 3. 5~10문제 (번역, 듣기, 말하기 혼합) → 4. 결과 + XP

### 색상 팔레트
- Primary Green: #58CC02
- Secondary Blue: #1CB0F6
- Background: #FFFFFF
- Correct: #58CC02 (초록)
- Incorrect: #FF4B4B (빨강)
- Text: #3C3C3C

## 3. 케이크 (Cake)

### 특징
- 영상 기반 학습: 실제 유튜브 영상 클립으로 자연스러운 표현 학습
- 따라 말하기: 음성 인식으로 발음 평가
- 핵심 표현 카드: 영상에서 추출한 핵심 표현을 카드로 제시
- 일일 학습 콘텐츠: 매일 새로운 영상 + 표현

### 색상
- Primary: #FF6B35 (오렌지-코럴)
- Background: 화이트
- 카드: 둥근 모서리(16px), 그림자 효과

## 4. 산타토익 (Santa TOEIC / Riiid)

### 특징
- AI 맞춤 학습: 사용자 수준 진단 후 개인화된 문제 제공
- 문제 풀이 UI: 지문 → 문제 → 선택지 순차 레이아웃
- 해설 화면: 정답 + 오답 이유 + 핵심 어휘 표시
- 진행률 대시보드: 예상 점수, 학습 통계

### 색상
- Primary: #6C5CE7 (보라)
- Accent: #00B894 (민트)
- Background: #F8F9FA

## 5. Grammarly

### 작문 교정 UI
- 인라인 교정: 작성 중 실시간으로 밑줄 표시 (빨강: 오류, 파랑: 스타일)
- 사이드 패널: 교정 제안을 카드로 나열, 클릭으로 적용
- 카테고리 분류: Correctness, Clarity, Engagement, Delivery
- 점수 게이지: 전체 작문 품질 점수 (0-100)

### 색상
- Primary Green: #15C39A
- Error Red: #E8344E
- Warning Yellow: #F0AD4E
- Suggestion Blue: #5AADFF

## 6. HiNative / LangCorrect

### 특징
- 커뮤니티 기반 작문 교정: 원어민이 직접 교정
- 교정 UI: 원문 ↔ 교정문 비교 표시, 취소선 + 삽입 형태
- 한글→영어 번역 요청 가능

## 우리 앱에 적용할 핵심 패턴

### 스와이프 카드 (말해보카 + 듀오링고)
- 좌우 스와이프로 문장 간 전환
- 카드 중앙에 한글 문장 크게 표시
- 하단에 진행률 인디케이터 (1/3, 2/3, 3/3)

### 힌트 시스템 (말해보카)
- 접기/펼치기 토글
- 영어 단어(Bold) + 한글 뜻(Light) 쌍

### 즉시 피드백 (듀오링고)
- 작문 완료 즉시 교정 결과 표시
- 정답 부분: 초록 하이라이트
- 수정 부분: 주황/빨강 하이라이트

### 교정 비교 (Grammarly)
- 원문(사용자 작문) ↔ 교정문 병렬 비교
- 수정 포인트별 설명

### 일일 루틴 (듀오링고)
- 하루 3문장 루틴
- 완료 시 축하 애니메이션 + 스트릭 표시

### 색상 전략
- 학습 집중을 위한 차분한 블루 계열 Primary
- 성공/정답: 초록 (#4CAF50)
- 수정 필요: 주황 (#FF9800)
- 오류: 빨강 (#F44336)
- 배경: 밝은 회색 (#FAFBFC) — 눈의 피로 최소화
