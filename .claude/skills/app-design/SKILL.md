# App Design — 영어 작문 앱 UX/UI 설계

영어 작문 학습 앱의 화면 구조, 컴포넌트 레이아웃, 스타일 가이드를 설계한다.

## 앱 컨셉

한글 문장을 보고 영어로 작문하면 LLM이 교정해주는 학습 앱. 하루 3문장, 일상영어/비지니스영어 테마.

## 벤치마킹 기준

디자인 시 반드시 `references/benchmarking.md`를 Read하여 참조 앱들의 UX 패턴을 확인한다.

## 스타일 가이드

### 색상 팔레트
- Primary: #4A90D9 (차분한 블루 — 학습 집중)
- Secondary: #7C4DFF (보라 — 힌트/강조)
- Success: #4CAF50 (초록 — 정답/좋은 점수)
- Warning: #FF9800 (주황 — 보통 점수)
- Error: #F44336 (빨강 — 낮은 점수)
- Background: #FAFBFC (밝은 회색)
- Card: #FFFFFF
- Text Primary: #1A1A2E
- Text Secondary: #6B7280

### 타이포그래피
- 한글 문장: Pretendard Bold 20px
- 힌트 단어 (영어): SF Pro Text Medium 16px
- 힌트 단어 (한글): Pretendard Regular 14px
- 작문 입력: SF Pro Text Regular 18px
- 교정 문장: SF Pro Text Medium 18px
- 설명 텍스트: Pretendard Regular 14px

### 간격
- 카드 패딩: 20px
- 섹션 간 간격: 16px
- 힌트 단어 간 간격: 8px
- 버튼 높이: 48px
- 카드 border-radius: 16px

### 인터랙션
- 스와이프: 좌우 스와이프로 문장 전환, threshold 40%
- 힌트 접기/펼치기: 탭으로 토글, 300ms 슬라이드 애니메이션
- 작문 완료 버튼: 작문 입력이 10자 이상일 때 활성화
- 교정 결과: 아래에서 올라오는 슬라이드 애니메이션 (400ms)
- 점수 표시: 숫자 카운트업 애니메이션
